# Act 3 pacing

- Favor consistency over greed: fewer explosive turns that require perfect draw sequencing.
- Save spike tools (potions, key powers) for Act 3 elites and the final boss path.
- Respect multi-attack patterns; weak is strong where it blocks multiple hits.
- Path to boss: prefer rest/recovery nodes if the deck lacks sustain for long fights.
