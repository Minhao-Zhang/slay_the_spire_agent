# Map pathing

- **Risk budget**: weigh path using HP, potions, and whether the deck answers upcoming elites.
- Elites and bosses are the main checks; hallways are partly fuel and setup.
- Act transitions are pivot points—adjust pathing if the deck shifted (more block, more scaling, etc.).
- Prefer paths with flexible forks when uncertain; dead-end risky lines punish bad reads.
