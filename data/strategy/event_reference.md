# Event decision patterns

- When an event has **3+** options, read each for immediate HP loss vs long-term payoff.
- Favor outcomes that fix a known hole (AoE, block, scaling) over raw gold if a hard fight is soon.
- Curses and max-HP trades: only if the deck can carry the downside or the upside solves a boss check.
- If HP is critical, prefer safe payouts and avoid gambling events unless potions/backline sustain exist.
