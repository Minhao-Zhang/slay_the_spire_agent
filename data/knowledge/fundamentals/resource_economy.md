# HP, gold, and shops

- **HP is currency**: spend it when ROI is clear (elite relic, path to shop); don’t bleed it for low-value fights. Healing is scarce—safer paths and potions beat marginal rewards.
- **Rest vs smith**: campfire **Rest** heals about **30% max HP**; **Smith** upgrades one card permanently. Rest when HP would lose the next line; smith when an upgrade hits a **breakpoint** (damage, block, draw) before a check. Merchant **removal** starts ~**75g** and rises **+25g** each time run-wide (relics can modify)—early removals spike deck density most.
- **Shops**: early removals fix draws before gear checks; late shops—buy what plugs the next elite/boss hole. Don’t skip a needed answer to hoard for a dream rare unless fights are solved.
- **Merchant priorities** (flexible): removal if bloated; rare/uncommon that solves the check; potion before a scary elite; then HP sustain if starved; shop relics can be run-defining when they match the deck.
